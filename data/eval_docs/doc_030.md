# 文档 31

西巴丹岛（）是马来西亚沙巴州著名岛屿，以优美的的自然景观及潜水环境闻名世界。，是马来西亚唯一的大洋岛，岛屿面积约4万平方米，从600米深的海底直接伸出海面。西巴丹岛在马来西亚成立前后即作为法定“鸟类禁猎区”予以保护。2009年沙巴州元首根据《公园法（1984年）》将该岛陆域部分（即全岛，约13.5公顷）公布为“西巴丹岛公园”；2015年环岛海域及珊瑚礁（约1万6846.5公顷）也经公告划入公园范围。2002年，海牙国际法庭就马来西亚、印度尼西亚两国对该岛和利吉丹岛主权争议一案判决两岛归属马来西亚。2000年4月，菲律宾恐怖组织阿布沙耶夫在该岛绑架了21人，包括10个游客及11个渡假村员工，人质被带到棉兰老岛，其后全部获释。
