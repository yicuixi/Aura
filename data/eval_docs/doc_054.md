# 文档 55

马岛灵猫（"Fossa fossana"），又名马尔加什灵猫、马岛麝猫或马岛缟狸，是马达加斯加特有的一种食蚁狸科。马岛灵猫以往与横带狸猫一同分类在缟狸亚科中，及后再分类在自己的马岛缟狸亚科中，但现已改在食蚁狸亚科中。牠们亦属于马岛灵猫属或马岛缟狸属（"Fossa"）中。马岛灵猫是细小的哺乳动物，体长约47厘米，尾巴长20厘米，重2.5公斤。牠们的外表及行动像狐狸。牠们的毛很短，呈灰色，由头部至尾巴有黑色的斑纹。这些斑纹在近腹部变成斑点。牠们的脚很短幼，但其爪是否可伸缩则有不同意见，没有肛门腺。牠们栖息在马达加斯加的热带森林及干旱落叶林。马岛灵猫是夜间活动的，是独居的或是成对生活的。牠们并非攀树能手，经常于山峡出入。牠们吃细小的脊椎动物、昆虫及鸟蛋。马岛灵猫的交配季节是介乎8月至9月间，妊娠期为3个月。幼猫较为发育完满，约10星期就断奶。由于马岛灵猫很稀少，故所知甚少。
