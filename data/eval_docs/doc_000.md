# 文档 1

半线天竺鲷，俗名目择仔、大目侧仔，为辐鳍鱼纲鲈形目鲈亚目天竺鲷科的其中一个种。该物种的模式产地在日本长崎县。本鱼分布于印度洋－西太平洋区，包括印尼、菲律宾、台湾、日本、澳洲北部等海域。水深3至100公尺。本鱼体稍长、侧扁，尾柄窄，头尖，眼径较吻长为长。第一背鳍小且和第二背鳍基底极近。尾呈叉状。颜色淡灰色，略带红，尾柄有和瞳孔同大小之圆黑点；第一背鳍硬棘末端呈漆黑色，体侧两条水平纵带间呈金黄色。背鳍硬棘8枚、软条9枚；臀鳍硬棘2枚、软条8枚。体长可达12公分。本鱼栖息在岩礁中，大多成群行动。夏季为产卵期，卵产下后，由雄鱼含于口中孵育。通常雄雌成对共同划定势力范围。以浮游生物和底栖无脊椎动物为食物。可食用，亦可作为观赏鱼观赏。
