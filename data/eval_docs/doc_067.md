# 文档 68

玉桂山（或）是位于香港南区鸭脷洲东部的山峰，高196米，是岛上的最高峰。山腰是利东邨、渔安苑和深湾轩，邻近海怡半岛、鸭脷洲邨和鸭脷排等地。玉桂山以西为鸭脷洲的另一山峰，可视为玉桂山的副峰，常被称为假玉桂山，又称佛爷山，海拔143米。1841年，查理·义律因英国不满他与清廷签署的《穿鼻草约》被召回伦敦，Alexander Robert Johnston始出任香港临时总督至翌年二月砵甸乍接任，玉桂山因纪念他而得英文名，此名与庄士敦无关。约1940年，英军在玉桂山上修建鸭巴甸炮台（），用作镇守香港南面海岸及南丫岛一带。及后为防止落入日军手上，炮台在香港保卫战期间由守军自行破坏。由于当时鸭脷洲的英文名为，因此炮台的名称取其英语音译为鸭巴甸炮台。战后玉桂山东坡被削走变成利东临时房屋区，成了利东邨的前身，后来重建成利东邨。2016年底港铁南港岛线通车后，该山因邻近港铁站利东站，加上一些传媒吹捧，而吸引大量游客到访，然而该山并无正式的登山径，只适合具一定经验的行山人士。玉桂山偶然也传出一些灵异传闻，约干年前有部份晨运客「听到」有军队的步操脚步声。
