# 文档 93

飞黄腾达7为美国真人秀《飞黄腾达》的第七辑，于2008年1月3日开始于美国NBC频道播出。本辑亦命名为名人版飞黄腾达（"The Celebrity Apprentice"，港译飞黄腾达 - 名人版）。香港无线电视明珠台于2008年6月7日起播放。本辑从第六辑的比赛地南加州的洛杉矶返回纽约市作出比赛。本辑的参赛者全是美国的知名人物（除Omarosa外），并为他们支持的慈善机构筹款，每集胜出的项目经理能为其机构筹得一笔款项，最后胜出者能为其机构获得25万美金。他们全部入住于纽约市的杜林普大厦（Trump Tower）。下表为本季度的参赛者，以其原队伍作出排列。队伍最初以当初的分组方法：男女各一组来分组。男方的队伍名为九头蛇（Hydra）（港译：许德拉），女方的队伍名为实业家（Empresario）（港译：企业家）。注：括号内的中文名字为香港（明珠台）译名
