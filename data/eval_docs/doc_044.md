# 文档 45

西土城站位于北京市海淀区，是北京地铁10号线的一个车站。西土城车站因位于元大都城垣遗址西北角，以及贯穿小月河两侧的西土城路而得名。西土城站位于海淀区知春路－北土城西路与学院路－西土城路相交的路口的南侧，在学知桥下，呈东西走向。10号线西土城站为了绕开路口南北向地下管线采用分离式站厅，车站东西两端为三跨框架结构采用明挖施工，中部为单层双联拱结构采用暗挖法通过，这是为了不影响南北向交通及地下管线。此外，本站是10号线一期设有公共艺术的10座车站之一。10号线西土城站设有1个岛式站台，位于知春路-北土城西路地底。西土城站拥有4个出入口：西北口 (A)、东北口 (B)、东南口 (C)、西南口 (D)，分别位于学知桥四角。2008年2月，10号线一期车站的站名确定，本站由学院路站更名为西土城站。同年7月19日，本站随10号线一期启用。2016年12月发布的昌平线南延段环境影响报告书显示，昌平线计划在该站与10号线换乘。
