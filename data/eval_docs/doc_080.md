# 文档 81

冠饰角龙类（学名：Coronosauria），是角龙下目的一个演化支，生存于白垩纪晚期，分布于现今的北美洲与东亚。冠饰角龙类与阎王角龙、黎明角龙、古角龙、辽宁角龙等物种，组成新角龙类。冠饰角龙类的头盾非常大，约占头骨长度的一半以上。前三节颈椎固定成单一个，可用来支撑大型的头部。喙骨与前齿骨形成狭窄的喙状嘴。顶骨与鳞状骨构成大型头盾。冠饰角龙类是由在1986年提出，范围是恐怖三角龙与安氏原角龙的最近共同祖先，与其最近共同祖先的所有后代。依照这个定义，冠饰角龙类包含纤角龙科、原角龙科、角龙科。但其内部分类仍有争议，根据不同的研究，纤角龙科比原角龙科原始，或是原角龙科比纤角龙科原始。
