# 文档 23

日本绿鹦嘴鱼，又名日本鹦哥鱼、火红绿鹦嘴鱼、红尾鹦嘴鱼，俗名鹦哥，为辐鳍鱼纲鲈形目隆头鱼亚目鹦哥鱼科的其中一种。本鱼分布于太平洋区，包括台湾、日本、菲律宾、印尼、马来西亚、新几内亚、澳洲、索罗门群岛、帛琉、密克罗尼西亚、斐济群岛、马绍尔群岛、东加、万那杜、新喀里多尼亚、马里亚纳群岛等海域。水深3至30公尺。本鱼体延长而略侧扁。头部轮廓呈平滑的弧型。齿板之外表面平滑，上齿板不完全被上唇所覆盖；每一上咽骨具1列臼齿状之咽头齿。开始型雌鱼，体为深红棕色，唯尾鳍为较淡的棕红色。胸鳍稍黄。齿淡红色。具稍圆的截形尾。终端型雄鱼，体深蓝绿色，各鳞片具一橘红色短斑。腹部色淡黄。颊部为鲑鱼红色，外缘镶著蓝线纹。背鳍及臀鳍均是黄色和蓝色纵斑。具截型尾。体长可达35公分。喜成群的生活于岩礁及珊瑚礁中，以珊瑚底藻为食。一般作为观赏鱼。可食用，食用时多用温火烧烤或清蒸。
