# 文档 90

胸斑眶锯雀鲷（学名："Stegastes fasciolatus"），又称蓝纹高身雀鲷、太平洋真雀鲷，俗名为厚壳仔，为辐鳍鱼纲鲈形目隆头鱼亚目雀鲷科的其中一种。本鱼分布于印度洋、太平洋海区，包括东非、马达加斯加、模里西斯、留尼旺、葛摩、马尔地夫、台湾、日本、菲律宾、马来西亚、印尼、新几内亚、澳洲、密克罗尼西亚、马绍尔群岛、马里亚纳群岛、索罗门群岛、夏威夷群岛、斐济群岛、万那杜、东加、吐瓦鲁、吉里巴斯、诺鲁、复活节岛等海域。水深0.5至10公尺。本鱼体呈暗褐色，腹侧颜色较淡，吻蓝色，眼下缘及吻后方亦各有一道蓝线，鳃叶后下方的鳞片上有蓝点，腹侧的鳞片上一多具有蓝点。尾鳍末端颜色较淡。鳃盖后方具2枚硬棘。幼鱼的背鳍和臀鳍有蓝边。背鳍硬棘13枚、背鳍软条15至17、臀鳍硬棘2枚、臀鳍软条12至14。体长可达12公分。本鱼栖息在沿岸水流较缓的岩礁区，具领域性，属于藻食性鱼类。多用以水族观赏，很少人食用。
