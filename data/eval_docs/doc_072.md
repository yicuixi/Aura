# 文档 73

美洲似银汉鱼（学名：）为辐鳍鱼纲银汉鱼目银汉鱼科的其中一种。本鱼分布于美国奥勒冈州、加州圣地牙哥至墨西哥下加利福尼亚沿岸。本鱼体呈长梭型，略侧扁，头短，鳍长略大于体高；吻尖突，两腭略等长；主上腭骨窄小，其末端不达眼前缘；前主上腭骨之上突起甚短，短于眼径；齿小而尖，并成小窄齿带，上腭之内列齿较大。背鳍两分离，第一背鳍较小，具9枚硬棘，其起点位于体背中央之略后；第二背鳍具1棘及12枚软条，臀鳍具1棘及23枚软条，其基底大于第二背鳍；胸鳍胸位，略与头长等长，末端不达腹鳍基底；腹鳍中腹位；尾鳍深开叉；体被细圆鳞，13纵列，一纵列约有鳞77枚。体淡绿色，体侧具一银色纵带，鳃盖黄色，体长可达45公分。常成群游泳于近岸，在岸边浅水域之鳗草上产卵，卵具黏著性，成团状。为重要的食用鱼，其肉白色，组织纤细，加州英语系居民成之为香鱼。FishBase
