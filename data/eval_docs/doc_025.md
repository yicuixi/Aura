# 文档 26

宋子仙（），侯景部将，侯景之乱时被杀。侯景之乱爆发，侯景自称汉王，以宋子仙为太保，任约为司空。太清三年（549年）十一月，侯景派宋子仙自吴郡攻钱塘（今浙江杭州南），戴僧逖投降。侯景闻江夏空虚，使任约、宋子仙率精骑四百偷袭，郢州刺史萧方诸被俘，鲍泉被丁和捶死。侯景入据江夏。宋子仙率军一万人，直攻巴陵（今湖南岳阳），侯景随后亦赶往巴陵，“缘江戍逻，望风请服”，非常得意。宜州刺史王琳随王僧辩驻巴陵，宋子仙久攻不克。此时侯景军中多病疫，又兼粮食告罄。六月，任约兵败于赤沙亭（今湖南华容南），被僧祐等俘获，六月三日，侯景闻知任约败，遂焚营退走，留下丁和与宋子仙以二万部队固守郢州（今湖北武汉）。大宝二年（551年）六月二十二日，宋子仙等困窘交集，向王僧辩投降，表示要献城，只求返回建康。王僧辩假意答应他，王僧辩趁宋子仙戒备松懈，命定州刺史杜龛率一千人攀上城墙，鼓噪奄进，发动攻击，宋子仙的军队大多溺死或被杀。宋子仙和丁和逃到白杨浦时被生俘。不久押送江陵斩首。
