# 文档 57

赤膀鸭（"Anas strepera"），俗称漈凫，是一种广泛分布的鸭。牠们最初是由卡尔·林奈（Carolus Linnaeus）于1758年描述。赤膀鸭长约46-56厘米，翼展78-90厘米。繁殖期的雄鸭呈灰色，形成一种图案，尾巴黑色，翼斑毛呈白色。非繁殖期时的雄鸭则像雌鸭。雌鸭呈浅褐色，像雌性绿头鸭。不过牠们的喙边呈深橙色，体型较为细小，而且没有明显的翼斑毛。赤膀鸭在欧洲和亚洲北部及北美洲中部繁殖。分布地扩展至北美洲东部。牠们是一种候鸟，迁往南方过冬。赤膀鸭原在很少会在英国出现，但近年开始有增加的趋势。估计是因牠们入侵当地，及在苏格兰等地繁殖。赤膀鸭生活在开放的塘地，如大草原及沙地、潮湿的草原或沼泽。牠们会将头潜入水中吃植物。牠们在地上筑巢，有时甚至会远离海边。牠们并非如其他钻水鸭般群居，倾向小群生活。牠们较为寂静，雄鸭的叫声沙哑，雌鸭会发出像绿头鸭的嘎嘎声。雏鸭最初是吃昆虫的，成鸭在繁殖期则有时会吃软体动物及昆虫。牠们是《非洲-欧亚大陆迁徙水鸟保护协定》下受保护的物种之一。
