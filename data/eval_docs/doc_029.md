# 文档 30

《大神一郎奋斗记》是日本世嘉 (SEGA)在其家用主机Dreamcast上推出的作品，被视为《樱花大战系列》中的一个外传游戏，亦被指为fan disc（较适合忠实粉丝游玩的作品）。内容以《樱花大战》花组的队长大神一郎为视点，讲述《樱花大战歌谣秀》中的舞台剧《红蜥蜴》公演前后，花组后台及帝都大剧场内所发生的事。游戏最大特点，是在适当时候穿插真人版《红蜥蜴》的公演片段，增加玩家置身在剧场内的真实感。游戏除了《樱花大战系列》一贯的LIPS冒险（在限时内选择对白）系统外，还设有数个迷你游戏，较特别的是玩家终可一尝大神一郎独家任务——在剧场内当剪票员的滋味。游戏流程十分短，而收集元素亦与正传系列不同，改为收集声优照片、真人版《红蜥蜴》剧照及海报照片等，而非收集游戏角色图片。
