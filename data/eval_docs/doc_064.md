# 文档 65

王欣逸，，台湾前男童星，因演出偶像剧《海豚湾恋人》中男主角童年时期达达一角而被大众所熟知。两位哥哥王骏逸、王鸣逸同为童星。出身隔代教养家庭，7岁为贴补家用出道，参与大爱等电视剧演出，因演出偶像剧《海豚湾恋人》而走红，之后偶像剧、电影、广告邀约不断，13岁时入围金钟奖。国中时加入竹联帮正堂，2012年8月14日曾因桃园警方扫荡竹联帮被捕。2016年7月21日深夜在桃园市芦竹区涉嫌帮叶姓友人出气，与同伴持铁铝棒、西瓜刀等物砍人，造成一名曲姓少年重伤昏迷，芦竹警方以车追人，7小时内就在新北市万里区被逮捕，7月25日，被依杀人未遂罪嫌声押获准，2016年11月3日，桃园地检署侦查终结，依杀人未遂罪嫌将王欣逸等5人提起公诉，目前已被交保候传。
