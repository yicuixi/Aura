# 文档 91

大鳞大眼鲷，又称日本锯大眼鲷，俗名红目鲢、红目眶，为辐鳍鱼纲鲈形目的其中一个种。本鱼分布于印度西太平洋区，包括东非、红海、台湾、澳洲、菲律宾等海域。水深80至200公尺。本鱼体呈赤红色，体甚高，显然侧扁，呈卵圆形。眼特大，瞳孔大半位于体中线下方。吻短。口裂大，近乎垂直；下颌突出，颌骨、锄骨和腭骨均具齿。前鳃骨后缘及下缘具锯齿并具有一枚后向之短强棘。头及体部皆被有粗糙坚实不易脱落之栉鳞，鳞后缘棘弱而多；侧线完全，侧线鳞孔数30至36枚。幼鱼体侧有5条白色之垂直条纹，较带间地色为狭，各鳍黑色，鳞片有短而宽之棘突。成鱼第二背鳍、臀鳍及尾鳍各鳍之末端有黑缘。背鳍具硬棘10棘，软条11至12枚；臀鳍与背鳍几相对，具硬棘3枚，软条9至10枚；背鳍及臀鳍后端圆形；胸鳍短小；腹鳍中长，短于或等于头长；尾鳍稍圆。体长可达20公分。本鱼栖息于岩岸之深海海底，靠近海底游泳，以捕食底栖生物如虾、蟹之幼生或贝类等为食，为夜行性鱼类。食用鱼，剥皮后红烧或其他吃法均可。
