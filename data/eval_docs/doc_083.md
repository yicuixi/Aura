# 文档 84

台北市万华区龙山国民小学，简称龙山国小，是台湾台北市万华区的市立小学。前身为「艋舺第二公学校」，于1919年4月1日假艋舺祖师庙创校，同日奉令指定为「台湾公立女子高等普通学校」（今中山女高）代用附属公学校。1921年随女子高等普通学校改制，再指定艋舺第二公学校为「台北州立女子高等普通学校」代用附属公学校。1922年，改「龙山公学校」，并同时再为改制的「台北第三高等女学校」附属公学校。1928年，终止三高女附属公学校，仍为龙山公学校。1941年，依「国民学校令」改「龙山国民学校」，校名于战后沿用。1968年台湾实施九年国民教育后，改为「台北市龙山区龙山国民小学」；1990年因台北市行政区重划，改「台北市万华区龙山国民小学」。认真学习<br>认真游戏诚：诚真实在<br>正：正大光明<br>勤：勤奋节俭<br>朴：朴实无华<br>龙山龙山你看龙山，<br>一龙叫起万龙动，<br>卷著瑞气满天红，<br>登龙门前香月桂，<br>龙山冠绝四海风，<br>你看啊你看啊，堂堂的团结阵，<br>哈哈哈哈，你听啊你听啊 ，<br>愉快的和乐颂，<br>龙山龙山龙山龙山，我们的龙山
