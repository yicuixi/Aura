# 文档 2

默哀，或称默祷，是指团体或个人为表示哀悼或吊唁，在一段时间内低头静立，视为一种礼仪，而非属任何宗教仪式。与半旗一样，默哀通常用于哀悼重要人物去世或是重大天灾或战争的死难者。习惯上默哀以分钟为基本单位，最常见的默哀时间是一分钟，具体时间则主要视乎哀悼对象的重要性而决定。在默哀的时间当中，一般是保持静默以示哀悼。但也有主持默哀者读秒计时的例子。中国古代就有默哀的纪录。例如唐代韩愈《谒衡岳庙遂宿岳寺题门楼》一诗中就有「潜心默祷若有应，岂非正直能感通」的内容。世界上最近一次大型默哀活动，是中华人民共和国国务院宣布于2008年5月19日北京时间14时28分，全国国民须为汶川大地震死难者默哀三分钟，香港特别行政区及澳门特别行政区按国务院公告随之。
