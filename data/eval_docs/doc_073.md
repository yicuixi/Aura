# 文档 74

美国特勤局（又译美国特勤处、美国秘勤局等，英文：United States Secret Service，缩写USSS）是美国联邦政府的执法机构，隶属于美国国土安全部。该机构雇员分为便衣特工和制服特工。2003年3月1日之前，特勤处隶属于美国财政部。1865年成立，为了秘密调查美元钞券与国债票券伪造案与其它金融犯罪，当时隶属美国财政部。1901年，美国总统威廉·麦金莱遇刺身亡，美国特勤局负有保护正副总统及亲属之安全的任务。1968年6月5日，正进行美国总统民主党内初选活动的罗伯特·F·甘迺迪参议员遇刺身亡后，总统参选人列入美国特勤局的保护范围。2001年911事件后，美国政府新成立国土安全部，特勤局改为该部管辖，不过负责任务不变。特勤局的主要管辖权是预防和调查对美元和国债债券与其它票券的伪造，以及保护以下人物:同时，该处还负责跟踪调查各种金融犯罪的相关嫌疑人，并为一些地方犯罪的调查提供协助。目前还有一些特勤处的任务还处于保密状态。负责保护各国驻美大使馆、美国海军天文台和在华盛顿特区的白宫。特勤局特工的着装很适合其职能。通常情况下，特工身着一套非常保守的西服，但依勤务需要亦可能身著燕尾服
