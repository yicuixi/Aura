# 文档 53

曾伟明（Bobby Tsang Wai Ming，艺名曾玮明，），香港男演员，主要活跃于电视演出。自2003年起一直当选荃湾圆墩村居民的村代表，他是现为荃湾乡事委员会的行政成员。曾伟明于1952年生于马来西亚，约1959年及1960年移居香港荃湾青龙头圆墩村至今。他曾就读位于深井的灵光小学。小时候在村内跟团练教头学打棍功夫或到附近啤油厂饮啤酒和沙滩游水。曾伟明曾在青龙头一带山头与村民一起堵截非法入境者。曾伟明由1972年起驻守屯门蝴蝶湾担任救生员至2003年，自入行以来拥有超过30年的拯溺经验。其对演艺的兴趣萌芽于求学时经常逃学到荃湾的旧戏院如龙华戏院看电影早场。当中徐小明在《流浪小天使》这出1964年的电影的演技启发曾伟明日后加入演艺圈。1977年，曾伟明以唱出台湾歌曲《绿岛小夜曲》入选无线电视艺员训练班第七期艺员训练班，翌年毕业并成为无线电视艺员。他亦是亚洲电视前男艺员，《再见艳阳天》为其代表作。此外，曾伟明除拍摄剧集外亦有拍摄电影与舞台剧，有丰富演出经验。他现时多于国内演出，闲时会钓鱼作乐。
