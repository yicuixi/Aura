# 文档 20

硕鬣狗（"Pachycrocuta"）是一属史前巨大的短面鬣狗。硕鬣狗最大及最多资料的短吻硕鬣狗站立时肩高约有1米，重达200公斤，差不多是一只细小狮子的重量，而牠却更为粗壮。牠们生存于300-50万年前的上新世晚期至更新世中期。在欧亚大陆、东非及南部非洲的多个位点都有发现牠们的化石。大部份遗骸都是碎片，一般是头颅骨，而在周口店亦有非常完整的硕鬣狗化石，估计牠们以洞穴为巢穴。硕鬣狗可能是成群狩猎大型动物，如鹿或更大，并会吃腐肉的。由于硕鬣狗较为大型，并不很适合长距离追捕猎物，故可能会较喜欢狩猎大型动物。这就与现今的斑鬣狗有所不同，因为斑鬣狗是较为灵活，有时会捕猎猎物，或是吃狮子吃剩的腐肉。硕鬣狗在生态位上与较细小（但仍是大型）的上新斑鬣狗，从来未有在同一区域发现这两类同期的生物。其他的硕鬣狗物种，如"P. robusta"及"P. pyrenaica"所知甚少。只知前者是棕鬣狗的欧洲极大型古亚种。"Hyaena bellax"则有时认为是属于硕鬣狗属的。
