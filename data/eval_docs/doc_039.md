# 文档 40

红钻鱼（学名：），又称大口滨鲷、红肉钻、红鸡仔、滨鲷，辐鳍鱼纲鲈形目鲈亚目笛鲷科的其中一个种。本鱼广泛分布于印度太平洋区，包括东非、红海、马达加斯加、波斯湾、模里西斯、塞席尔群岛、波斯湾、马尔地夫、巴基斯坦、印度、安达曼海、缅甸、泰国、马来西亚、越南、菲律宾、中国、日本、琉球群岛、澳洲、台湾、印尼、澳洲、马里亚纳群岛、密克罗尼西亚、所罗门群岛、诺鲁、帛琉、斐济群岛、万纳杜、新几内亚、新喀里多尼亚、夏威夷等海域。水深90至400公尺。本鱼体延长呈纺锤形，标准体长约为体高之3.35倍。两眼间隔平扁。眼前方无沟槽。下颌突出于上颌；上颌骨末端延伸至眼中部的下方；上颌骨被鳞。上下颌具多列细齿，外列齿扩大；上下颌前端具2至4犬齿；锄骨具窄的弧状齿带。体被中大型栉鳞，背鳍及臀鳍上均裸露无鳞；侧线完全且平直。鱼体为鲜艳的玫瑰红，腹侧为浅红色或银白色，背鳍、尾鳍、吻、颔、眼、口内、鳃孔内皆为红色，背鳍硬棘10枚，软条11枚；臀鳍硬棘3枚，软条8枚；胸鳍长约等于头长；尾鳍深分叉，下叶比较长。其明显特征为沿著侧线有条黄色纵带自鳃盖后方延伸到尾鳍基部。体长可达127公分。本鱼栖息在深沟、大凹坑或斜坡地，离底5
