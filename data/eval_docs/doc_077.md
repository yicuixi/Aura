# 文档 78

张静美是台湾的漫画家，出生于台北市，稻江家职（稻江护家前身）家政科毕业，代表作为《代战天使》、《青春小鸟》。19岁时（1980年）进入伊士曼小咪出版社（大然文化前身）所举办的小咪漫画培训班，并以〈生命回响曲〉获得第1届小咪漫画新人奖第2名；翌年再以〈水仙子〉获得第2届小咪漫画新人奖第1名。21岁时（1982年）以〈灵香园传奇〉于伊士曼《小咪漫画周刊》开始连载而正式出道，22岁时（1983年）以〈可家少爷〉于《汉堡漫画》开始连载，23岁时（1984年）以〈如梦似幻〉获得民国73年度全国漫画大擂台连环漫画奖佳作，24岁时（1985年）以〈杜鹃．杜鹃〉于时报文化《欢乐漫画半月刊》开始连载。25岁时（1986年）以〈哑妻〉于时报《欢乐漫画半月刊》开始连载，该作品改编自言情小说家琼瑶所著小说《六个梦》；隔年（1987年）本作品并获得国立编译馆优良漫画奖佳作。26岁时（1987年）以〈我心飞舞〉于时报《欢乐漫画》开始连载，同年以〈奇梦神话〉于《汉堡漫画》开始连载，期间转到大然《公主快报别册》继续连载。27岁时（1989年）以短篇〈祝你快乐〉、〈泪莲〉刊载于《汉堡漫画》，28岁时（1990年）以短篇
