# 文档 34

新加坡鹦嘴鱼，又名绿颔鹦哥鱼，俗名鹦哥、青衫（雄）、蚝鱼（雌），为辐鳍鱼纲鲈形目隆头鱼亚目鹦哥鱼科的其中一种。本鱼分布于印度西太平洋区，包括马尔地夫、斯里兰卡、台湾、菲律宾、马来西亚、印尼、新几内亚、澳洲东岸等海域。水深10至30公尺。本鱼开始型雌鱼，体棕红色，各鳞片具淡色缘。身体布有蓝色小斑点，尤其颊部更具有小斑点结合成蓝色短斑纹。终端型雄鱼，体深蓝绿色，各鳞片具淡色缘。腹部色淡。吻部及颊部形成淡蓝绿色的不规则斑。背鳍深蓝绿色相间有棕黄色的垂直窄斑纹。雌鱼及雄鱼均为不明显的双凹形尾。背鳍硬棘9枚、背鳍软条10枚、臀鳍硬棘3枚、臀鳍软条9枚。体长可达56公分。喜成群地栖居于陡峭的珊瑚礁斜坡水域。以藻类及底栖生物为食。食用鱼，食用时可清蒸、红烧或盐烧。另外可作为观赏鱼。
