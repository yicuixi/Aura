# 文档 41

海宁西站，是沪昆客运专线的一个车站，位于浙江省嘉兴市海宁市许村镇报国村沈士大道以西。车站于2010年10月26日随沪昆客运专线沪杭段开通而启用，由上海铁路局嘉兴车务段管理。沪杭客运专线在规划之初并未在海宁设立站点，但嘉兴市政府以该线在本地区里程最长、占用土地面积最多为由，向铁道部、上级政府积极奔走，最终争取到在海宁设立站点。车站站房于2010年3月开工。2010年10月26日车站随沪杭客运专线开通而启用，首班停靠的列车为开往的G7417次列车。海宁西站总建筑面积1.7万平方米，其中站房面积4,933平方米。站房以海宁钱江潮为意向，插入了粉墙黛瓦的江南民居元素，由浙江省一建建设集团有限公司建设。站台位于站房南侧，高于候车室6米，设置侧式站台2座，到发线2条（不含2条正线）。海宁西站距离海宁市区超过30公里远。海宁西站开通后，原先停靠海宁站的动车组列车全部停运。因当时杭州东站尚未完工，所有海宁站的普速列车除了停靠杭州站仅有的4班列车外，所有列车全部停靠位于萧山区的杭州南站，导致前往杭州的海宁市民非常不便。在沪杭高铁上，桐乡站反而比海宁西站更靠近海宁市区，而海宁亦开通了往桐乡站的T188路公
