# 文档 47

成年礼或者成人礼，是指人类为了庆祝其成员长大成人者而举行之礼俗仪式。中国古代成人礼指冠礼和笄礼，西周就已存在，一直延续到明朝。周朝贵族男子满20岁时举办冠礼，即加冠，之后可以娶妻。加冠指男性被族群承认，表示其已成人。如果是公室皇室，加冠年纪可以再提前，因为：年幼的帝王若迟迟未行冠礼，会让臣民觉得他还是个孩子，在统治上有所不便。春秋时鲁襄公就是12岁加冠的例子。周朝贵族女子则是在满15岁举办笄礼，及笄之后可以许嫁。若女子提早许嫁，也可提早行笄礼。若女子到了二十岁都还没许嫁，也应行笄礼，因此时她已是完全成熟的女子，不再适合做女孩的打扮。成年礼也有定在满十六岁时举办者，如日本人的元服礼、潮州人的出花园以及台湾人的做十六岁。日本现代的成年礼是在20岁，在大学进行；除了已满20岁的女生们穿著和服，男生则穿西装进行典礼，祭典后也会喝酒（日本成人年龄）。除此以外，家人也会把礼品放在时间木盒；而它将会保持至少5年，最长时间为青年男女的儿女成为20岁后，得由当年出席的所有成员（即青年人和他/她的父母和兄弟姐妹）在场下开启。如果在特定时间内，任何人因健康（重病、意外）或婚姻理由（父母离婚、再婚）而确定不能
