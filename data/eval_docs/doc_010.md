# 文档 11

一号风球（），是澳门特别行政区的热带气旋警告信号，是热带气旋警告信号之中最低级的信号，高一级是三号风球。受热带风暴卡努影响，最近一次一号风球于2017年10月13日下午9时半悬挂。为一戒备信号，指有热带气旋中心集结在澳门特别行政区800公里范围内，而其情况足以于稍后可能影响澳门。与香港的三号强风信号不同（2007年之后），当在三号风球除下后，澳门地球物理暨气象局甚少以一号风球取代三号风球。而强烈热带风暴帕布是首次及惟一一次令气象局以一号风球取代三号风球。检查排栅、棚架、花盆、天线等可被风吹毁的物品是否稳妥，细小船只应逗留于避风塘附近。除额外注明外，以下纪录由2000年开始计算。2008年的台风浣熊，由4月17日16:15至4月18日20:45（共28小时半）。而2000年前则是1967年的台风维奥莉，在4月9日悬挂。2000年的强烈热带风暴贝碧嘉，由11月4日16:00至11月8日09:00（共97小时）。而2000年前则是1974年的台风艾玛，在12月1日悬挂。2000年的强烈热带风暴玛莉亚，由8月28日10:00至9月1日13:00（共99小时）。2007年的强烈热带风暴帕布的回马
