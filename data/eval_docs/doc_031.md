# 文档 32

索靖（），字幼安，晋朝敦煌人。家族世代为官。晋朝官员及书法家。从小就气质非凡，聪颖过人，与同乡泛衷、张甝、索紾和索永皆进太学，并称敦煌五龙。索靖博通经史，又通纤纬之学。后在凉州作别驾，而一次朝廷命郡举出紧良方正之士，接受朝内高官的考问政事和经义，索靖就被推举并到洛阳，傅玄和张华与索綝见过面后都隆重的去结交他。索靖后拜驸马都尉，又出外任驻守西域的戊己校尉长史。因著在朝内任官的同郡人张勃推举而被召入朝，担任尚书郎。在朝同为尚书郎的潘岳、罗尚和顾荣都对他十分佩服。多年后，索靖先后出任雁门太守、鲁相及酒泉太守。至永熙元年（290年）晋惠帝即位，索靖获赐爵关内侯。元康六年（296年），北地郡的马兰羌及卢水胡皆叛乱，索靖转任征西大将军司马肜的左司马，加荡寇将军，屯兵粟邑参与平叛，曾击败叛胡。后改拜始平内史。永康二年（301年），赵王司马伦篡位称帝，遭到齐王司马冏、成都王司马颖及河间王司马颙三王为首的讨伐。当时索靖亦响应义举，获授左衞将军。司马伦倒台后，索靖以讨伐孙秀之功获加授散骑常侍。及后迁后将军。太安二年（302年），河间王司马颙举兵讨伐在朝内掌政的长沙王司马乂，索靖被其任命为使持节、监洛城诸军
