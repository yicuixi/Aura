# 文档 71

海山汽车客运股份有限公司，简称海山客运，曾是大台北地区公路客运经营者之一，原总部位于台北县三峡镇。海山客运的前身为三峡汽车运输合作社，也是今日中兴巴士集团旗下之淡水客运前身。1974年3月，三峡汽车运输合作社由于经营困难，让售于台北客运，并更名为三峡汽车客运股份有限公司（简称三峡客运）。1977年，基隆颜家买入台北客运与三峡客运，因三峡客运所经营之路线大部分以1950年县辖区建制废除前的台北县海山区为主，故把三峡客运更名为海山客运。海山客运被基隆颜家购入后，将位于三峡镇的办公地点并入当时位于板桥市文化路的台北客运总公司（现在已经搬迁至三重区）。1980年代末期，海山客运的路线已经大部份委交台北客运代驶，难以见到海山客运车辆。1991年，海山客运正式并入台北客运，相关路线的营运权亦转移之。
