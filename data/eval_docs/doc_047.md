# 文档 48

秋山幸二（あきやま こうじ、1962年4月6日－）为日本的棒球选手之一，曾经效力于太平洋联盟西武狮、福冈大荣鹰队，守备位置为外野手。于2009年～2014年担任褔冈软体银行鹰的总教练。1980年以240万日圆加盟西武狮队，签约金5000万日圆为当年最高。在1987年获得了单季的全垒打王。1989年达成3成打击率并有30轰30盗的纪录，为日本职棒第五人达成。隔年更达成了30支全垒打・50次盗垒成功的优异纪录。于西武狮时期，协助球队获得了8次联盟优胜、6次日本第一。在1991年年的日本职棒总冠军战，由西武狮对决广岛鲤鱼的战役当中，七战中轰出了四发全垒打，并获到了该年总冠军战的最高殊勋选手赏。而后移籍到大荣鹰，仍成为大荣鹰的中心主砲，1999年、2000年球队于太平洋联盟封王，也是贡献甚钜。其中大荣鹰在1999年击败中日龙获得日本第一，再度获得1999年总冠军战的最高殊勋选手赏。在2000年8月，也达成了2000支安打的纪录。于2002年退休，并接任软银鹰二军监督一职，2008年底王贞治卸下一军监督一职，钦点秋山幸二接任一军监督。担任软银鹰监督时期共率领球队夺得2011年、2014年两次的总
