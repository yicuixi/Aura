# 文档 50

岩手宫城内陆地震发生于2008年6月14日日本当地时间8时43分（协调世界时23时43分），震央位于岩手县奥州市和宫城县栗原市，即距离岩手县县厅盛冈市西南60公里。根据日本气象厅的数据，此次地震的日本气象厅地震规模为7.2。至7月14日的8时00分，震度1以上的余震有511次。其中最大余震就是6月14日9时20分发生的M5.6的地震。根据日本气象厅所公布的地震正式名称为平成20年（2008年）岩手・宫城内陆地震（英文名称："The Iwate-Miyagi Nairiku Earthquake in 2008"）。截至2009年6月9日，统计出这次地震造成的损失如下：死者死亡原因分别为：落石2名，山泥倾泻3名，泥石流5名，山泥倾泻导致车辆埋没1名、受到地震惊吓跑出车道而发生交通事故1名；还有2人死亡原因不明日本气象厅通过2007年10月始开始投入使用的全国性地震预警系统（紧急地震速报）发出预警，避免了严重人员伤亡。
