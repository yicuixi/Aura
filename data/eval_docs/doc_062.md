# 文档 63

约瑟夫·查尔斯·弗朗西斯·洛克（Joseph Francis Charles Rock，），美籍奥地利探险家、植物学家、地理学家和语言学家。曾于20世纪初，以美国《国家地理杂志》、美国农业部、哈佛大学植物研究所的探险家，撰稿人、摄影家的身份到云南滇缅边境以及西藏考察。1884年生于奥地利维也纳，1907年移居美国夏威夷檀香山，并在此成为植物群落专家。1911年进入夏威夷大学，主要从事植物学研究工作，建立了当地第一所植物标本馆，并在1911-1920年间担任馆长。1913年获得美国国籍。1920年离开夏威夷大学，启程前往亚洲进行探险考察。洛克的植物探索旅程启航于泰国、缅甸和印度阿萨姆地区，最初旨在找寻大风子树。从1922年到1949年，“植物猎人”洛克在中国云南、四川、甘肃东南以及西藏东部度过了漫长的探险考察岁月，对当地植物群落、人文风俗等多个方面进行了深入考察，并将多种植物样本带回西方，今天位于波士顿南部的阿诺德植物园保留了许多这一时期采集的植物样本。此后，洛克以丽江附近的村落为基地，对当地风土人情和动植物都进行了考察研究。这一时期的许多文章，都刊于国家地理杂志上。也正是这一时期的文章
