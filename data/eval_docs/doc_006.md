# 文档 7

1993年的香港，以罕见而严重的人群践踏事故揭开序幕，酿成21死62人轻重伤，国际传媒亦有广泛报道。年中又发生严重工业意外，一架地盘升降机因超重及缺乏保养维修而直坠平台，造成12人丧生，成为香港史上第二严重的建筑业意外。中英关系则因为末代香港总督彭定康于1992年10月突然推出直选成份极高的政改方案而陷入谷底，中英双方虽经过多轮政制谈判，惟谈判破裂，中方决定「另起炉灶」（即后来成立的临立会），于1995年经过直选选出的立法局议员将不能够直接过度至1997年香港回归后，而国务院港澳事务办公室主任鲁平对彭定康的一句「千古罪人」则成为名句。本年香港经济畅旺，股市屡创新高，由年初约5,000点逐步攀升至年底的11,000点，升幅超过一倍。年内，亦有多位知名人士逝世，2月，著名香港粤语片演员吴楚帆在温哥华病逝。5月，立法局议员张鉴泉因为突发心脏病去世；6月，著名唱作歌手、摇滚乐队BEYOND的主音黄家驹于出席日本电视台游戏节目时失足堕台，经过多番救治后返魂乏术，终年31岁，死讯轰动香港社会，出殡期间数千人在烈日下追逐灵车，场面混乱；10月，电视艺员余绮霞因鼻咽癌逝世，终年36岁。于80年代红极一
