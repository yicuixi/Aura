# 文档 70

王金璐（），京剧演员，工生行，马连良弟子，中国国家1级演员，国家级非物质文化遗产京剧项目代表性传承人。出身北京中华戏曲学校「金」字科，1934年经校长焦菊隐推荐，与师兄王和霖同拜马连良先生门下。曾任上海京剧院及其前身华东戏曲研究院实验京剧团演员，得周信芳院长许多指点。1979年起任北京中国戏曲学院表演系教授。2002年得到中华人民共和国文化部造型表演艺术创作研究成就奖，同时得奖的还有红线女、袁世海等艺术家。2008年1月26日，他和李金泉、张春华、谭元寿、赵燕侠、杜近芳、梅葆玖、刘秀荣、杨秋玲、李世济、孙毓敏、刘长瑜、李长春、李维康、叶少兰、张幼麟、李荣威、周仲博、汪庆元、尚长荣、陈少云、王梦云、孙正阳、关栋天共同列进中华人民共和国文化部第2批国家级非物质文化遗产项目代表性传承人名单。
