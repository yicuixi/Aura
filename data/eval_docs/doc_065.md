# 文档 66

北海道狼，又称虾夷狼，是一种已灭绝的狼，曾分布在日本的北海道，以及俄罗斯的萨哈林州（桦太、千岛群岛）一带，后来因为人类大量猎杀，导致北海道狼的数量减少，在1889年灭绝。北海道狼和分布在本州、四国、九州的日本狼为不同亚种之近亲，日本狼亦于1905年灭绝。日本曾经有两种狼，一者是大家都叫熟悉的日本狼，是最小的狼。分布于本州、四国及九州。另一种是虾夷狼，又称北海道狼，顾名思义，这种狼居住在北海道地区，和日本狼是不同亚种的近亲。两种狼都曾经被当做守护神崇拜，有过供奉日本狼的祠庙，其民间传说也不少。而在北海道，虾夷狼则被爱奴族称为「远吠之神」。但当日本渐渐现代化后，这些「かみさま」也不被尊崇，虾夷狼最先灭绝，而日本狼也在1905年于奈良消失。
