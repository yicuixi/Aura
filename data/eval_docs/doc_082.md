# 文档 83

虹彩吸蜜鹦鹉（学名："Trichoglossus haematodus"）又称彩虹鹦鹉、五彩绯胸鹦鹉、绿颈吸蜜鹦鹉。颜色鲜艳，拥有多达21个亚种。体长25-30厘米，体重120-130克。分布在印度尼西亚、东帝汶和澳大利亚东部、北部等地。此外，自20世纪初，虹彩吸蜜鹦鹉也发现在香港和华南地区繁殖。虹彩吸蜜鹦鹉聪敏机灵，不但能学会各种把戏，而且语言能力也相当不错。野生状态下，它们生活在低地森林等处，成群结队，快速飞行。自然，它们的食物主要为花蜜、花粉,也吃植物种子、嫩叶、昆虫。18-20月时公鸟成熟，母鸟则需两年。十二月份为其发情期，但一般全年都可交配。每窝产卵两到三枚，孵化期25天，羽化期7-8周。虹彩吸蜜鹦鹉属于鹦鹉科，包含在分类学的鹦形目之下。虹彩吸蜜鹦鹉常被人认为包含了red-collared lorikeet("T. rubritorquis")这个亚种，但今日大多数的专家论点认为两者是独立开来的。此外，在1997年的一个复审中建议将小巽他群岛上一些最有特色的吸蜜鹦鹉分开来做为独立品种，也就是scarlet-breasted lorikeet ("T. forsteni")、m
