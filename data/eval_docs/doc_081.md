# 文档 82

食肉牛龙亚科（Carnotaurinae）是兽脚亚目阿贝力龙科的一个亚科。食肉牛龙亚科是由美国古生物学家保罗·塞里诺（Paul Sereno）在1998年提出，范围是：阿贝力龙科中，接近食肉牛龙，离阿贝力龙较远的物种。这个范围包含奥卡龙（阿根廷）、食肉牛龙（阿根廷）、玛君龙（马达加斯加）、胜王龙（印度）。2008年，卡纳尔等人南美洲食肉牛龙亚科的系统发生分析。这个分析指出，所有来自南美洲的食肉牛龙亚科（包括肌肉龙属）都属于食肉牛龙亚科下的次演化支，这个演化支名为短吻龙类，意为“短的吻部”。卡纳尔等人把短吻龙类定义为：离萨氏食肉牛龙较近，但离凹齿玛君龙较远的阿贝力龙科。而由托尔托萨等人于2013年所做的分析则是把数个属于食肉牛龙亚科的属重新归类为玛君龙亚科，并把数个阿贝力龙科归类为食肉牛龙族。
