Aura AI 文档

个人智能助手开发与使用指南

入门指南

欢迎使用Aura AI - 您的个人智能助手。Aura融合了多种先进AI技术，可以通过命令行或Web界面与您交互。

核心功能

  • 自然语言交互与个性化对话
  • 基于您个人知识库的智能问答
  • 长期记忆与偏好学习
  • 文件操作与网络搜索能力

快速开始

  1. 运行 start_aura.bat 启动命令行界面
  2. 运行 start_webui.bat 启动Web界面
  3. 在命令行或OpenWebUI中开始与Aura对话
查看安装说明

安装说明

按照以下步骤安装和配置Aura：

前置条件

  • Python 3.10+
  • Docker 和 Docker Compose
  • 8GB+ RAM (建议16GB+)

基础安装

# 安装依赖
pip install -r requirements.txt

# 启动Docker服务
docker-compose up -d

模型下载

# 下载Qwen模型
docker exec -it aura_new ollama pull qwen3:4b

初始化设置

# 初始化知识库
python manage_knowledge.py --init

# 启动Aura
python aura.py

OpenWebUI 使用指南

访问界面

在浏览器中访问 http://localhost:3000

初始登录

  • 用户名: admin
  • 密码: admin

连接模型

首次使用时，需要连接到Ollama服务：

  1. 点击"连接模型"
  2. 设置Ollama URL为 http://aura_new:11434
  3. 点击"连接"

如需更详细的指南，请参阅 OpenWebUI完整指南

API参考

Aura API端点

端点 方法 说明
/api/system_status GET 获取系统状态
/api/search_knowledge POST 在知识库中搜索
/api/save_memory POST 保存记忆
/api/recall_memory POST 回忆记忆
/api/search_web POST 网络搜索
/api/read_file POST 读取文件
/api/write_file POST 写入文件
/api/add_document POST 添加文档到知识库

请求示例

// 搜索知识库
fetch('http://localhost:5000/api/search_knowledge', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ query: '什么是Aura?' })
})

故障排除

常见问题

Docker容器无法启动

检查Docker服务是否运行，以及端口是否被占用：

docker ps -a
netstat -ano | findstr "3000 8088 11435"

模型加载失败

检查Ollama容器日志：

docker logs aura_new

OpenWebUI连接问题

确保Docker网络配置正确，检查docker-compose.yml中的网络设置。
